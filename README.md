mbsis\_project



mbsis\_project — это учебно-практический Django-проект бухгалтерской информационной системы, разработанный в соответствии с принципами Турецкого Единого плана счетов (Tek Düzen Hesap Planı).



Проект ориентирован на моделирование реальной корпоративной бухгалтерии с периодным учётом, журналом операций и формированием финансовой отчётности.





Основные возможности

\- Ведение бухгалтерских счетов (Accounts)

\- Журнал проводок (Yevmiye)

\- Строки журнала с автоматической нумерацией

\- Учёт по финансовым периодам

\- Формирование:

&nbsp; - Оборотно-сальдовой ведомости (Mizan)

&nbsp; - Баланса (Balance Sheet)

&nbsp; - Отчёта о прибылях и убытках (Income Statement)

\- ER-диаграмма структуры данных



Используемые технологии

\- Python

\- Django

\- SQLite (для разработки)

\- HTML / Django Templates

\- Git \& GitHub

Запуск проекта локально

```bash

cd mbsis\_project



Создание виртуального окружения

python -m venv venv



Windows

venv\\Scripts\\activate



macOS / Linux

source venv/bin/activate



Установка зависимостей

pip install -r requirements.txt



Применение миграций
